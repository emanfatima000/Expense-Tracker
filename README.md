# Expense Tracker

A Python-based Expense Tracker built using Object-Oriented Programming (OOP) and File Handling.

## Features

- Add new expenses
- View all expenses
- Search expenses by category
- Remove expenses
- Save expenses to a text file
- Load expenses from a text file

## Technologies Used

- Python
- Object-Oriented Programming (OOP)
- File Handling

## How to Run

Run the following command:

```bash
python expense_tracker.py
```
